# Warehouse Management System 📦

A robust Java-based console application designed to streamline inventory tracking, manage product stocks, and handle warehouse operations efficiently using Object-Oriented Programming (OOP) principles.

## 🚀 Features

- **Add New Products:** Easily register items with unique IDs, names, and initial stock quantities.
- **Stock Management:** Dynamically update stock levels when products are added or removed.
- **Inventory Overview:** Displays a structured and clean list of all available items in the warehouse.
- **Interactive Menu:** Simple, user-friendly console-driven interface for easy navigation.

## 🛠️ Core Concepts Demonstrated

- **Object-Oriented Programming (OOP):** Classes, Objects, and Encapsulation.
- **Data Structures:** Efficient use of data storage for inventory tracking.
- **Algorithm & Logic:** Structured conditional statements and loops for inventory filtering.
